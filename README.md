# LyncFellow

LyncFellow is an open source project by Glueck & Kanja. LyncFellow makes i-Buddy compatible to Microsoft Lync.

## Features

  * i-Buddy's head glows in the color of your status
  * i-Buddy dances on incoming phone calls
